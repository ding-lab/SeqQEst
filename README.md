# SeqQEst (Sequence Quality Estimate)

```
Estimate sequencing data (WES/RNA-Seq) quality

QC-L1 (quality control level 1) - check sequnecing quality
QC-L2 (quality control level 2) - detecte swap or outgroup samples
```

## Install

```
picard
samtools
fastqc
7z
bamreadcount

python3.7
```
